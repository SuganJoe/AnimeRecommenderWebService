To execute the project:

1. Open the AnimeRecommenderWebservice.sln in Visual Studio
2. In AnimeRecommenderWebService right click on -> Reco_Service.asmx, choose "View in Browser". 
3. Once the SOAP service is running, launch the Website.
4. In the UI, input any username. (e.g., jack) and click on the 'Search' button
5. The recommendations for the user and the user's watch history is displayed.
